2、变量的作用域

要理解闭包，首先必须理解Javascript特殊的变量作用域。

变量的作用域无非就是两种：全局变量和局部变量。

Javascript语言的特殊之处，就在于函数内部可以直接读取全局变量。

另一方面，在函数外部自然无法读取函数内的局部变量。

这里有一个地方需要注意，函数内部声明变量的时候，一定要使用var命令。如果不用的话，你实际上声明了一个全局变量！

有时候需要得到函数内的局部变量，在函数的内部，再定义一个函数。

在上面的代码中，函数f2就被包括在函数f1内部，这时f1内部的所有局部变量，对f2都是可见的。但是反过来就不行，f2内部的局部变量，对f1 就是不可见的。这就是Javascript语言特有的“链式作用域”结构（chain scope），子对象会一级一级地向上寻找所有父对象的变量。所以，父对象的所有变量，对子对象都是可见的，反之则不成立。

既然f2可以读取f1中的局部变量，那么只要把f2作为返回值就可以在f1外部读取它的内部变量了。

3、使用闭包的注意点

1）由于闭包会使得函数中的变量都被保存在内存中，内存消耗很大，所以不能滥用闭包，否则会造成网页的性能问题，在IE中可能导致内存泄露。解决方法是，在退出函数之前，将不使用的局部变量全部删除。

2）闭包会在父函数外部，改变父函数内部变量的值。所以，如果你把父函数当作对象（object）使用，把闭包当作它的公用方法（Public Method），把内部变量当作它的私有属性（private value），这时一定要小心，不要随便

改变父函数内部变量的值。

闭包可以用在许多地方。它的最大用处有两个，一个是前面提到的可以读取函数内部的变量，另一个就是让这些变量的值始终保持在内存中。

flex定义：

布局的传统解决方案，基于盒状模型，依赖 display 属性 + position属性 + float属性。它对于那些特殊布局非常不方便，比如，垂直居中就不容易实现。
Flex 是 Flexible Box 的缩写，意为”弹性布局”，用来为盒状模型提供最大的灵活性。任何一个容器都可以指定为 Flex 布局。
注意，设为 Flex 布局以后，子元素的float、clear和vertical-align属性将失效。
.box{
display: flex;
}

.box{
display: inline-flex;
}

flex基本概念：

采用 Flex 布局的元素，称为 Flex 容器（flex container），简称”容器”。 它的所有子元素自动成为容器成员，称为 Flex 项目（flex item），简称”项目”。

容器默认存在两根轴：水平的主轴（main axis）和垂直的交叉轴（cross axis）。
主轴的开始位置（与边框的交叉点）叫做main start，结束位置叫做main end；
交叉轴的开始位置叫做cross start，结束位置叫做cross end。

项目默认沿主轴排列。单个项目占据的主轴空间叫做main size，占据的交叉轴空间叫做cross size。

在评论上看到有人说《爱乐之城》剧情老套，当然，我不否认。

我们太过于习惯于将电影的各个要素分割开去评价一部电影，而忽略了一个事实——电影的各个方面是相互依存的，取其中任何一部分是无法独立存在的。若是想评论剧情，就只能结合配乐、人物形象、情感表现等其他方面综合起来分析。

剧情的优劣完全可以左右一部的电影品质好坏，然而老套的剧情本身并不坏，是我们自身的问题。我们之所以认为某部电影剧情老土，只是因为我们看了太多这样的故事发展，倘若是世界上第一个这样写的人，那必定会一鸣惊人吧。《追忆似水年华》能把主人公入睡前在床上的辗转反侧这样一个平常的场景写得如此细致，打的当然不是剧情牌。

试想当我们只看到《当幸福来敲门》是一个普通职员有着不幸的经历，经过一番奋斗终于成功的励志故事，试想我们只看到《爆裂鼓手》严师出高徒的套路剧情，那么自然我们定会认为确实不怎么样。若是这样，那么电影的真正魅力将无法展现出来。世界上有太多的“老套”电影，其实说来说去这个世界上的剧情也就那么多，新颖的剧情固然更容易受人喜爱，但“老套”的故事也并不是毫无光彩。

我想说，一部令人称赞的电影，创作它的人通过各种各样的修饰美化，再平常的剧情也可以成为佳片。而爱乐之城正是做到了这一点。电影本身是否足够优秀看的是综合实力。

《爱乐之城》延续了达米安•查泽雷的“霓虹风格”，再加上颜色鲜艳明亮的服饰造型，颇有复古之感。

现在，结合剧情，谈谈《爱乐之城》中与那些美化故事的亮点

本片最大的亮点即是——音乐（原声带LaLaLand）

音乐的介入使得剧情得以升华，内容也不再单调，当然若是不喜欢音乐或者只喜欢流行音乐的人另当别论。来说作为一部优秀的歌舞片，《爱乐之城》的有着杰出的配乐，而音乐响起的同时也意味着观众情绪的调动。

电影的开场以洛杉矶拥堵的交通为背景，超长镜头拍摄，上百人在高架桥上纵情的舞蹈。在《Another day of sun》欢快明亮的旋律中，开始叙说这美丽的洛城爱情故事，也揭开了“乐”与“城”的序幕。这一段舞蹈有种看MV的感觉。整齐划一的舞姿，热情洋溢的脸庞，这座城市仿佛充满了活力，音乐和洛杉矶再也分割不来了。所有人跳完之后恢复拥堵的交通之后，镜头向下一转，女主角Mia坐在汽车里，对着长长的车队按喇叭。男主角不停按着喇叭。然后男女主角第一次照面，女主就竖了个中指。

Mia是个想要成为演员的咖啡师，参加过多次试镜，却每每不如意。（不知道那些人怎么想的，明明又漂亮又演的好）一次又一次的试镜失败让Mia难过不已。Mia在一次试镜失败后和她的同居女友们去参加派对。

电影的又一个亮点——色彩。胶片拍摄的全片色彩选取凸显出复古的情调，令人眼前一亮。大量的室内、夜景、街灯等电影场景，暗色调加灯光，这也是达米安•查泽雷导演的一贯风格，但相比《爆裂鼓手》，《爱乐之城》在色彩与灯光的运用上更为新颖大胆。色彩与灯光交映，每个观众的内心都变得梦幻起来。

Mia和她的小伙伴们从住处一直到派对结束都伴随着《Someone in the crowd》这首歌，整个派对的过程，导演通过一段漂亮的蒙太奇压缩成短短的一分多钟，既不累赘啰嗦，又可以融合进歌舞元素，足以体现导演的才智与造诣。然而绝不仅仅如此，在这一段的蒙太奇中，包含了太多有关Mia的信息。透过杯中的酒水我们可以窥见，Mia与派对的一切格格不入，无尽的娱乐只会让她厌倦。通过穿透酒水这样一个视觉，观众很容易感受到派对的梦幻与躁动，这正是影片中的绝妙之处。尽管派对的气氛再欢快，Mia总是无法融入那样的世界，她是孤独的，她在渴求某样东西。

《爱乐之城》中多次利用舞蹈来推动情节，Mia与Seb的情感穿插其中，与舞蹈融为一体。不重在描写事情发展的外观，而在于表现舞蹈而透出本质。这是导演相当巧妙的表现主义手法。

2017-08-31

花若盛开 清风自来


文/平台部 陈励平

　　花若盛开，蝴蝶自来；你若精彩，天自安排。你若盛开，笑靥如花，清风自来，如沐春风，我心安好，便是晴天。

　　春去秋来，该走的留不住，是我的赶不走，当学会坦然应对这些得失的时候，我想我便可静静地待以春暖花开时，而我笑靥如花。

　　接受现在和过去的模样，美好和精彩的未来自然会来。作为一个刚步入社会的大三女生，多少都会有这样那样的不适应，离开了安全的庇护所，独自来到一个陌生的城市，虽充满了期待也伴有担忧。因为所有的未知都在等着我。那时的我难以接受那时的模样。

　　去学会更多，知道自己其实可以承受更多。铜筋铁骨自然会为你撑起一片天。从不会到会，我学会了如何找工作，怎么去应对面试，怎么去租房，怎么去安抚自己的情绪。或许在大家看来这并不是什么大问题，而对于我，就是难。当我数百份简历石沉大海，我内心十分焦灼不定，感觉时间的每一分每一秒都在拷问我的无能，怀疑我的选择，几天里我没有多余的话，也没有好的睡眠，郁结于心，难以自持，害怕家里的电话让我的小宇宙爆发。想过逃避，我直接回家过暑假好啦！干嘛这么为难自己。所幸我没有，找到了一份还不错的工作，感觉还不错。

　　生活和天气一般无二，穿插各种插曲，沉着面对，免疫的大伞自然张开。第一天上班我坐错公交，错过了最佳上车时间。穿着高跟鞋，那天的太阳也很烈，站在十字路口，不知道往哪里走。走到一个很荒的地方，旁边都是裸露着钢筋混凝土的高楼，还有一处长满荒草的土地，没有看到公交也没有看到出租，看着导航所指的路，只能过车，一想到自己今天必然迟到有可能丢掉工作，自己就觉得特别无助，特别委屈，后来我还是坚持走了那条逆行的不停传来喇叭声警示的很窄的路，终于找到了公交站，眼看就要到了，要坐的车从我眼前开走了！当时没忍住，泪水止不住地流了下来。但事实上，事情并没有我想象得那么乌云密布，只是我自己把事情过于放大了。现在想想，我自己都笑了。

　　如果你花开盛夏，清风自来。不懂花的泥土不必在意。一切似乎都稳定了下来，生活步入正轨，正常上下班，晚上还能和异地的男朋友谈一谈近况。我构想着一切都是那么美好。可是尽不如人意，我在慢慢失去我三年的感情，因为第一次异地，互相不理解，不体谅。最终不欢而散。我最纯粹最珍贵的初恋，难忘而又不舍，我还没有做好任何准备，就失去了。当晚，我表现得很平静，因为不知道该怎么面对。第二天周一我照常上班，假装忘记一些事情。但是周末，再也忍不住了，我哭了，哭得还很大声。任我怎么伤心，怎么难过，什么也改变不了，除了折磨自己没有别的，后来我明白，没有什么是不变的，有些东西或许从来就不是自己的。即使曾经拥有。学会拿得起放得下也就释然了。你走我不留，你来我不阻。

　　人的心情就跟风中飞舞的铃铛一样；一头是天空，代表希望与梦想，另一头是心底，代表表情和旋律。你越安静，越能听清它的声音，你若越是喧嚣，反倒越听越恼，风起，音來；缘生，相守；淡然，如风；清脆，如铃，仅此而已。心情美了，花自然开了，清风也就来了。

　　每个人一生都在追求属于自己的盛夏，真正实现了的，或者是还在追求的人，在等待花开的时候一样艰难，坦然接受和面对才能更加沉着冷静。花开绚烂，蝴蝶翩翩，清风扑面，我付之一笑。

　　蝉之一夏，畅鸣无憾。而我要花之一生，尽美无暇，与清风相生相伴。
